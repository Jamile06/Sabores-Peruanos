## Descripción
Esta aplicación es una página web de un restaurante peruano diseñada para gestionar reservas de forma sencilla. Permite a los clientes conocer el restaurante, ver los horarios, consultar la carta de platos y realizar reservas. La interfaz es moderna, intuitiva y completamente responsive.
